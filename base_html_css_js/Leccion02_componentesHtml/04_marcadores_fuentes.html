Marcadores fuetne HTML

B: Se utiliza para poner en negrita un texto.

BIG: Se utiliza para agrandar una fuente, pero es interpretado por cada navegador,
por lo que el resultado puede ser diferente.

I: Se usa para poner en cursiva un texto.

S: Se usa para tachar un fragmento de texto.

STRIKE: Misma función que "S", tacha un texto.

SMALL: Se utiliza para hacer más pequeña una fuente, pero es interpretado por cada navegador,
por lo que el resultado puede ser diferente.

TT: Se utiliza para formatear como un teletipo, como en los
monitores o impresoras antiguas.

U: Se utiliza para subrayar un fragmento de texto.
